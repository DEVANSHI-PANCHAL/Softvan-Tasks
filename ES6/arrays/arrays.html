array methods:
  • push: adds element at the last index
  • pop: removed element at the last index
  • shift: removes the first element and returns item
  • unshift: adds element to the beginning and returs new array length
  • delete: array elements can be deleted using delete operator
  • concate: used to join two arrays
  • sort: arranges the elements in increasing(alphabetically) order
  • Splice: used to add new items in an array, returns deleted items and modifies an array
  • slice: slices out a peice from an array , it creates a new array
  • reduce: reduces an array to single value
