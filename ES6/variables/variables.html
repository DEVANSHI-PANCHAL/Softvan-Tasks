Variables

  • variables are containers used for storing data
  • js is dynamically typed
  • js allows to change the value of variable at run time
  • letters digits and numbers are allowed in declaring variables but not at the beginning
  • reserved keywords are not allowed for declaring variables
  • variables are case sensitive

diff btw let and var and const

  1. var is globally scoped while let and const are block scoped
  2. var can be updated and redeclared within its scope
  3. let can be updated but not re declared
  4. const cannot be updated nor redeclared
  5. var variables are initialised with undefined whereas let and const are not
  6. const must be initialised during declaration unlike let and var
